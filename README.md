# Pacific Lightning

`Pacific Lightning` is a browser-playable vertical shooter inspired by classic late-80s and early-90s arcade warplane games.

## Included

- P-38 Lightning player craft with fast, responsive movement
- enemy fighter waves and heavier attack craft
- battleship strike targets
- recurring boss encounters
- three power-up types: spread fire, beam cannon, and shield restore
- bomb clear mechanic for crowd control
- sprite-sheet driven player, enemy, boss, ship, projectile, explosion, and HUD art

## Play

Open `index.html` in a browser.

Controls:

- `WASD` or `Arrow Keys`: move
- `Space`: fire
- `Shift` or `X`: bomb
- `Enter`: start or restart

## Assets

Game art is stored in `assets/` and loaded directly by the browser version of the game.
